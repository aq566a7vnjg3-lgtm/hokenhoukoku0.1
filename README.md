# 営業入力・管理アプリ（Apps Script版）

## 概要
- 営業マンがスマホ / iPad から入力
- 管理者が一覧確認・絞り込み・件数集計
- 保存先は Google スプレッドシート

## 入力項目
- 拠点番号（プルダウン）
- 担当名（手入力）
- 提案件数（数字）
- 証券回収（数字）
- 獲得件数（数字）
- 区分（新規 / 保有期）

## 管理画面機能
- 日付絞り込み
- 拠点番号で絞り込み
- 担当名検索
- 合計表示
- 件数集計
- 重複判定（重複未確認 / 重複あり / 重複なし）

## セットアップ手順
1. Google スプレッドシートを新規作成
2. メニューの「拡張機能」→「Apps Script」を開く
3. 初期の `Code.gs` をこの ZIP の `Code.gs` 内容に置き換える
4. `Index.html` を新規 HTML ファイルとして作成して内容を貼り付ける
5. `appsscript.json` も置き換える
6. 保存する
7. Apps Script の「デプロイ」→「新しいデプロイ」→「種類を選択」→「ウェブアプリ」
8. 実行ユーザーは自分、アクセス権は運用に合わせて設定
9. デプロイ後のURLを開く

## シート構成
デプロイ後、初回アクセス時に `entries` シートが自動作成されます。
